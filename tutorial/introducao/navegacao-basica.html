🧭 Navegação Básica

Domine a interface e navegue com facilidade

📱

Anatomia da Interface

A interface Auzap.ai é dividida em 3 áreas principais:

1. Barra Superior (Navbar)

  • Logo Auzap.ai - Clique para voltar ao dashboard
  • Notificações 🔔 - Alertas de mensagens e eventos
  • Perfil 👤 - Configurações da conta e logout
  • Theme Toggle 🌙 - Alterna entre modo claro/escuro

2. Menu Lateral (Sidebar)

  • 📊 Dashboard - Visão geral e métricas
  • 💬 Conversas - Inbox de mensagens
  • 👨‍👩‍👧‍👦 Clientes & Pets - CRM completo
  • 📅 Agendamentos - Calendário e reservas
  • 📦 Catálogo - Serviços e produtos
  • 📊 Analytics - Relatórios detalhados
  • 🤖 IA Config - Configurar assistente
  • ⚙️ Configurações - Preferências gerais

3. Área de Conteúdo

É onde todo o conteúdo principal aparece. Muda conforme o menu selecionado. Sempre responsiva e otimizada para mobile e desktop.

⌨️

Atalhos de Teclado

Use atalhos para navegar mais rápido:

Busca Global Ctrl + K
Ir para Dashboard Ctrl + H
Ir para Conversas Ctrl + M
Nova Conversa Ctrl + N
Toggle Menu Lateral Ctrl + B
🔍

Busca Global

A busca global (Ctrl + K) permite encontrar rapidamente:

  • 📱 Conversas por nome ou número de telefone
  • 👤 Clientes por nome ou email
  • 🐕 Pets por nome ou raça
  • 📅 Agendamentos por data ou serviço
  • 📄 Páginas e funcionalidades da plataforma
💡
Dica Pro: A busca é inteligente! Digite parcialmente e ela encontra resultados relevantes.
📱

Versão Mobile

No mobile, a interface se adapta automaticamente:

  • 🍔 Menu lateral vira menu hamburguer
  • 📊 Cards do dashboard ficam em coluna única
  • 💬 Chat ocupa tela inteira quando aberto
  • 👆 Gestos de swipe para navegar rapidamente
⚠️
Nota: Algumas funcionalidades avançadas funcionam melhor no desktop. Recomendamos usar computador para configurações iniciais.
